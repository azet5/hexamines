## Build
For all platforms you have to [install Rust](https://rust-lang.org/tools/install/).

### Unix-like system
Install development packages using package manager.
- `libsdl2-dev`
- `libsdl2-image-dev`

To build, open directory with source code in terminal and run:
```text
cargo build --release
```
